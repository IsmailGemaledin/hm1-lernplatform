Farben und Hintergründe

Diese Lektion behandelt CSS-Techniken zur Farbgestaltung von Texten, Hintergründen und Elementen.

1. Farbdefinitionen

CSS unterstützt verschiedene Methoden, Farben anzugeben.
  • Schlüsselwörter: red, blue, green
  • Hexadezimal: #RRGGBB, z.B. #ff0000
  • RGB-Funktion: rgb(255,0,0)
  • RGBA-Funktion (mit Alpha/Transparenz): rgba(255,0,0,0.5)
  • HSL-Funktion: hsl(0,100%,50%)
p {
    color: blue;
}

div {
    background-color: #f0f0f0;
}

2. Hintergrundbilder

Hintergründe können auch mit Bildern gestaltet werden.
  • background-image: url('bild.jpg');
  • background-repeat: no-repeat | repeat-x | repeat-y;
  • background-size: cover | contain | 100px 200px;
  • background-position: center | top | bottom | left | right;
body {
    background-image: url('background.png');
    background-repeat: no-repeat;
    background-size: cover;
    background-position: center;
}

3. Farbverläufe (Gradients)

CSS kann lineare und radiale Farbverläufe als Hintergründe darstellen. 
div {
    background: linear-gradient(to right, red, yellow);
}

section {
    background: radial-gradient(circle, blue, white);
}

4. Transparenz und Opazität

Die Transparenz von Elementen wird mit opacity oder RGBA-Werten gesteuert. 
div {
    background-color: rgba(0, 128, 255, 0.5);
}

p {
    opacity: 0.7;
}

5. Textfarben

Die Textfarbe wird mit color festgelegt. 
h1 {
    color: #2b7cff;
}

p.note {
    color: orange;
}

6. Zusammenfassung

  • Farben: Schlüsselwörter, Hex, RGB(A), HSL
  • Hintergründe: Farbe, Bild, Gradients
  • Positionierung und Wiederholung von Hintergrundbildern
  • Transparenz: RGBA oder Opacity
  • Textfarbe über color
Die Beherrschung von Farben und Hintergründen ist zentral für ästhetisches und benutzerfreundliches Webdesign.
