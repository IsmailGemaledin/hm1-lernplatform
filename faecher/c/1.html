Einführung in C

Dieses Kapitel führt in die Programmiersprache C ein. Ziel ist es, die Grundidee, Struktur und Denkweise von C zu verstehen.

1. Was ist C?

Definition.
C ist eine prozedurale, kompilierte Programmiersprache, die in den 1970er-Jahren entwickelt wurde. Sie ist nah an der Hardware, effizient und bildet die Grundlage vieler moderner Systeme (z.B. Betriebssysteme, Embedded Systems).
  • sehr schnell
  • manuelle Speicherverwaltung
  • klare, einfache Syntax
  • Basis vieler anderer Sprachen (C++, Java, Python intern)

2. Aufbau eines C-Programms

Jedes C-Programm besitzt eine main-Funktion. Das Programm startet immer dort.
Beispiel – kleinstes C-Programm: 
#include <stdio.h>

int main(void) {
    return 0;
}
Erläuterung:
  • #include <stdio.h> bindet Standard-Ein/Ausgabe ein
  • int main(void) ist der Programmeinstieg
  • return 0; signalisiert erfolgreiches Beenden

3. Ausgabe mit printf

Die Funktion printf wird verwendet, um Text oder Werte auf der Konsole auszugeben.
Beispiel: 
#include <stdio.h>

int main(void) {
    printf("Hallo Welt!\n");
    return 0;
}
  • \n erzeugt einen Zeilenumbruch
  • Anführungszeichen sind Pflicht für Text

4. Variablen und Datentypen

Eine Variable speichert einen Wert eines bestimmten Datentyps.
  • int – ganze Zahlen
  • double – Kommazahlen
  • char – einzelne Zeichen
Beispiel: 
#include <stdio.h>

int main(void) {
    int a = 5;
    double b = 3.14;
    char c = 'A';

    printf("a = %d\n", a);
    printf("b = %f\n", b);
    printf("c = %c\n", c);

    return 0;
}

5. Rechnen in C

C unterstützt die üblichen arithmetischen Operatoren: +, -, *, /. 
int x = 10;
int y = 3;

int sum = x + y;
int prod = x * y;
int quot = x / y;   // ganzzahlige Division
Bei int-Division wird **abgeschnitten**, nicht gerundet.
