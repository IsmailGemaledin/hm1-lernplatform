Listen und Aufzählungen

In dieser Lektion lernen wir, wie man strukturierte Listen in HTML erstellt und formatiert, einschließlich geordneter, ungeordneter und verschachtelter Listen.

1. Ungeordnete Listen (Bullet Points)

Ungeordnete Listen werden mit <ul> erstellt. Jeder Listeneintrag steht in <li>. 
<ul>
    <li>Apfel</li>
    <li>Banane</li>
    <li>Kirsche</li>
</ul>

2. Geordnete Listen (Numbered)

Geordnete Listen werden mit <ol> erstellt und nummerieren automatisch die Einträge. 
<ol>
    <li>Schritt 1</li>
    <li>Schritt 2</li>
    <li>Schritt 3</li>
</ol>

3. Verschachtelte Listen

Listen können ineinander verschachtelt werden, um Unterpunkte zu erstellen. 
<ul>
    <li>Obst
        <ul>
            <li>Apfel</li>
            <li>Banane</li>
        </ul>
    </li>
    <li>Gemüse</li>
</ul>

4. Listentypen mit CSS

Die Darstellung der Listenpunkte kann mit CSS geändert werden:
  • list-style-type: disc | circle | square | decimal | lower-roman | upper-roman;
  • Beispiel für ungeordnete Liste: list-style-type: square;
ul {
    list-style-type: square;
}

5. Zusammenfassung

  • Ungeordnete Listen: <ul> + <li>
  • Geordnete Listen: <ol> + <li>
  • Verschachtelung für Unterpunkte möglich
  • Listentypen können mit CSS angepasst werden
