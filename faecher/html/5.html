Links und Hyperverweise

Diese Lektion behandelt die Erstellung von Hyperlinks in HTML, deren Typen, Zielattribute und Best Practices.

1. Grundlegender Hyperlink

Ein Hyperlink wird mit dem <a>-Element erzeugt. Das Attribut href enthält die Zieladresse. 
<a href="https://www.example.com">Besuche Example</a>

2. Relative und absolute Pfade

Hyperlinks können absolut (vollständige URL) oder relativ (bezogen auf die aktuelle Seite) sein.
  • Absolute URL: https://www.example.com/page.html
  • Relative URL: ./seite.html oder ../ordner/seite.html
<a href="./kontakt.html">Kontakt</a>

3. Öffnen in neuem Tab/Fenster

Mit target="_blank" öffnet sich der Link in einem neuen Tab oder Fenster. 
<a href="https://www.example.com" target="_blank">Externe Seite öffnen</a>

4. Titel-Attribut

Das title-Attribut zeigt beim Überfahren mit der Maus zusätzlichen Text. 
<a href="https://www.example.com" title="Zur offiziellen Website">Example</a>

5. E-Mail-Links

Mit mailto: kann ein Link zum Öffnen eines E-Mail-Programms erstellt werden. 
<a href="mailto:info@example.com">E-Mail senden</a>

6. Anker-Links (Sprungmarken)

Anker-Links springen innerhalb derselben Seite zu einem bestimmten Element mit einer id. 
<a href="#abschnitt1">Zu Abschnitt 1</a>

<h2 id="abschnitt1">Abschnitt 1</h2>

7. Zusammenfassung

  • Grundelement: <a href="URL">Text</a>
  • Relative und absolute Pfade
  • Öffnen in neuem Tab: target="_blank"
  • Zusätzliche Hinweise: title
  • E-Mail-Links: mailto:
  • Anker-Links: Sprung innerhalb der Seite
