Arrays und Strings in C++

Diese Lektion behandelt Arrays und Strings in C++. Ziel ist, die Speicherung von Daten in zusammenhängenden Speicherblöcken und die Arbeit mit Text zu verstehen.

1. Arrays

Ein Array ist eine Sammlung von Elementen desselben Typs, die in aufeinanderfolgenden Speicherplätzen liegen. 
int zahlen[5]; // Array von 5 ints
zahlen[0] = 10;
zahlen[1] = 20;
// Zugriff auf Elemente über Index (0-basiert)

2. Initialisierung von Arrays

Arrays können bei Deklaration initialisiert werden. 
int zahlen[5] = {1, 2, 3, 4, 5};
double werte[] = {0.1, 0.2, 0.3}; // Größe wird automatisch erkannt
char buchstaben[3] = {'A', 'B', 'C'};

3. Zugriff und Iteration

Elemente eines Arrays werden über den Index angesprochen. Schleifen sind ideal zur Iteration. 
for (int i = 0; i < 5; i++) {
    std::cout << zahlen[i] << " ";
}
std::cout << std::endl;

4. Mehrdimensionale Arrays

Arrays können mehrere Dimensionen haben, z.B. für Matrizen. 
int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
std::cout << matrix[1][2] << std::endl; // Ausgabe: 6

5. Strings in C++

In modernem C++ werden Zeichenketten bevorzugt mit std::string aus der Standardbibliothek verwaltet. 
#include <string>
#include <iostream>

int main() {
    std::string name = "C++ lernen";
    std::cout << "Text: " << name << std::endl;
}

6. Grundoperationen mit Strings

Strings unterstützen viele nützliche Operationen. 
std::string s1 = "Hallo";
std::string s2 = "Welt";
std::string s3 = s1 + " " + s2; // Verkettung
std::cout << s3 << std::endl;

std::cout << "Länge: " << s3.length() << std::endl;
std::cout << "Erstes Zeichen: " << s3[0] << std::endl;

7. Char-Arrays als klassische Strings

Historisch wurden Strings als nullterminierte char-Arrays verwendet. 
char name[] = "C++";
std::cout << name << std::endl;
Die Länge eines Char-Arrays muss mindestens das Endezeichen '\0' enthalten. Modernes C++ bevorzugt std::string.

8. Zusammenfassung

  • Arrays speichern mehrere Werte desselben Typs zusammenhängend im Speicher
  • Index beginnt bei 0
  • Mehrdimensionale Arrays für Matrizen und Tabellen
  • std::string ist die bevorzugte Art, Strings zu handhaben
  • Strings unterstützen Verkettung, Zugriff auf Zeichen, Länge und viele Methoden
  • Char-Arrays sind historisch, aber std::string ist sicherer und komfortabler
